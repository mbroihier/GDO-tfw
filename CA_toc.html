Controller Authentication

This test category verifies requirements associated with the authentification of the byte stream received by the controller from the "user interface".

  1. CA-0010 - A valid byte pattern
  2. CA-0020 - A second valid byte pattern
  3. CA-0030 - A third valid byte pattern, but it will have an extra byte
  4. CA-0040 - A fourth valid byte pattern, but it will have a missing byte from the end
  5. CA-0050 - A random byte pattern
  6. CA-0060 - A valid byte pattern after a large buffer overflow random set of bytes
  7. CA-0070 - Free form Bytes Test - too few - one byte
  8. CA-0080 - Free form Bytes Test - too few - two bytes
  9. CA-0090 - Free form Bytes Test - too few - three bytes
  10. CA-0100 - Free form Bytes Test - too few - four bytes
  11. CA-0110 - Free form Bytes Test - five bytes, bad pattern
