Test Case COMMAND-0030
Title If the controller is off, the UI will display a communication error when the button is pressed
Objective

This test case is intended to verify the following implied or specified requirements:

  • When the command button is pressed, the app shall:
  • Wait for and accept a controller reply.
  • If the reply is not received within an acceptable time, a Communication Error status message shall be displayed.

By having the controller off after running the last two test cases, this test case will demonstrate that the user interface will display a communication failure message because the controller will not be accessible.
Setup
  • Log into the controller and power off the bluetooth device.
Actions
Expected The user interface status field will display "Communication Failure".
Results UNTESTED
Cleanup
  • Log into the controller and power on the bluetooth device.
Requirements GDO-PR-UI-030, GDO-PR-UI-030.003, GDO-PR-UI-030.003.002
