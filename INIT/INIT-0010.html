Test Case INIT-0010
Title Power on Initialization of the Garage Door Controller does not move the garage door
Objective

This test case is intended to verify the following implied or specified requirements:

  • After power application to the controller, the controller shall:
  • Start a bluetooth server.
  • If a power cycle occurs, the return of power itself shall leave the power relay open. This will prevent the door from opening simply due to a power cycle.

By plugging in the controller to a power source and then confirming the garage door does not move, this test case will demonstrate that cycling power will not inadvertently open the garage door.
Setup
  • Connect the controller to the relay and the relay to the garage door opener.
Actions
  1. Plug the controller into a power source.
  2. Verify that the garage door does not move
Expected When powered up, the controller will not command the garage door opener to do anything.
Results UNTESTED
Cleanup
Requirements GDO-PR-CONTROLLER-010, GDO-PR-CONTROLLER-010.002, GDO-PERF-030
