Test Case INIT-0035
Title Bluetooth Server is Started
Objective

This test case is intended to verify the following implied or specified requirements:

  • After power application to the controller, the controller shall:
  • Initialize the push button connection to the garage door opener to open (no button pressed).

The successful execution of this test case will demonstrate that the controller implements a bluetooth server using the RFCOMM protocol and communicates to a client.
Setup
  • Assumes test case INIT-0010 has been executed and that the controller is powered.
Actions
  1. Using the Android interface, press the "OPEN/CLOSE" button.
  2. Verify that the the status message displays "Processing....".
  3. Verify that the status message then switches to either "Command Successful" or "Authentication Failure".
Expected A bluetooth RFCOMM server will be started that will respond to requests from a bluetooth RFCOMM client.
Results UNTESTED
Cleanup
Requirements GDO-PR-CONTROLLER-010, GDO-PR-CONTROLLER-010.001
