Test Case INIT-0015
Title UI Initialization
Objective

This test case is intended to verify the following implied or specified requirements:

  • The GDO user interface shall have one button that will be used to command the door open or closed.
  • The GDO user interface shall display a momentary status message after the button is displayed that shall display the status of the command attempt.
  • The GDO user interface shall display an authentication failure message if it receives an authentication failure response from the Raspberry PI 0 W.

Successful execution of this test case verifies that the user interface displayed on the Android device has the expected fields and buttons.
Setup
  • Bluetooth on Android device is disabled.
Actions
Expected A status field and command button will exist. When the command button is pressed when bluetooth is disabled on the Android device, a communication failure message will be displayed in the status field.
Results UNTESTED
Cleanup
Requirements GDO-UI-EXT-IF-010, GDO-UI-EXT-IF-020, GDO-UI-EXT-IF-020.002
