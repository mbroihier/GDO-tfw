Test Case PERF-0010
Title Response Time is Less than 5 Seconds
Objective

This test case is intended to verify the following implied or specified requirements:

  • From button press to status display update shall take no longer than 5 seconds.

Given that the controller and user interface are up, the time from button push to status display update will be an acceptable interval less than five seconds.
Setup
  • Controller is up and operational.
  • The user interface is displaying the main menu with status field and command button.
Actions
  1. From the Android user interface device, press the "OPEN/CLOSE" button.
  2. Verify that the time it takes from button press to the status update is less than 5 seconds.
  3. Repeat this for a total of 10 trials.
Expected When the button is displayed, the time interval from pressing the button to seeing the controller response will be less than 5 seconds.
Results FAILED
Cleanup
Requirements GDO-PERF-010
