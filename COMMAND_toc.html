Commands

This test category is intented to verify requirements associated with the end to end processing of an open/close command. These test cases specificially verify that the garage door opens and closes confirming that the bluetooth and electricial connections are correct.

  1. COMMAND-0010 - Pressing button opens door
  2. COMMAND-0020 - Pressing button closes door
  3. COMMAND-0030 - If the controller is off, the UI will display a communication error when the button is pressed
